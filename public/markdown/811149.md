## Description
> 식물이 되어가는 독으로 가득 찬 세상. 이제, 모든 숲은 인간을 죽인다. 인간들이 사막으로 밀려 난지 1000년. 인간들은 사막화 장치로 보호 받는 드넓은 사막 속 도시에 살고 있다. 지구를 구할 수 있는 유일한 방법이 숲 안에 있는 마녀에게 있다는 정보를 얻은 생태계 복구 기관<Roots>. 그들은 마녀를 찾아내기 위해 숲으로 향한다.
