# 가인의 표

## Author

## Description
> 이상하게도 사람들에게 배척받는 가인. 늘 혼자인 가인에게 괴물은 속삭인다. "신에게 버림받은 불쌍한 내 친구, 우리의 원수가 사랑하는 세상을 피 흘리게 하자!" 가인은 괴물의 제안을 받아들일까? [낙인. 내 이마에는 죄인의 낙인이 있다.] 이마의 표식이 있는 가인과 소리를 잃어버린 세상에서 가인이 만난 소년.
